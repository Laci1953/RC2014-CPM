Edit the following files: TE.H, DYNM.AS to choose the target hardware

M128 : 2 x 64KB RAM
M512 : 512KB RAM

SC108 : set it to 1 for 128KB RAM systems
MM : set it to one for Phillip Stevens 32KB EPROM + 128KB RAM module
SC114 : set it to 1 for SC114 
S80 = Simple80 : set it to 1 for Simple80 

Then:

SUBMIT MAKETE
SUBMIT LINKTE

For Z80ALL, see https://github.com/Laci1953/Software-for-Z80ALL-retro-computer/tree/main/TE
